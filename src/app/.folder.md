# Folder: /src/app

1. **地位**：Next.js App Router 页面层，处理路由和页面渲染。
2. **逻辑**：接收用户请求，调用 /lib 获取数据，渲染页面。
3. **约束**：页面组件应保持轻量，复杂逻辑抽离到 /lib。

## 成员清单
- `page.tsx`：首页
- `layout.tsx`：全局布局
- `globals.css`：全局样式
- `/blog/`：博客相关页面
- `/reading/`：阅读清单页面
- `/works/`：作品展示页面
- `/about/`：关于页面
- `/api/`：API 路由
- `sitemap.ts`：SEO sitemap
- `robots.ts`：SEO robots
